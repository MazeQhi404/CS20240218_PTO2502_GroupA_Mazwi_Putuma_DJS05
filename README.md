# DJS05: Show Detail Page with Routing and Navigation

# Project Overview:

A responsive podcast browser with dynamic routing and state preservation.

## Features
- Dynamic show pages 
- Search + genre filtering (persists on navigation)
- Season accordion with episode details
- Loading & error states
- Fully responsive

## Setup

npm create vite@latest DJS05-Podcast-App -- --template react
cd DJS05-Podcast-App
npm install react-router-dom
npm run dev
